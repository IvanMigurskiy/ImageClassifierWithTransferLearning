# Image Classifier Repository
Этот репозиторий содержит скрипт для обучения и оценки модели классификации изображений с использованием transfer learning. В нем реализованы функции для подготовки данных, построения модели, обучения, оценки и сохранения модели.

## Описание

Этот проект предназначен для быстрого запуска обучения модели на ваших данных, организованных по указанной структуре каталогов. После запуска скрипт автоматически предложит ввести параметры обучения, и модель начнет тренироваться, показывая прогресс и метрики. Также реализована функция автоматического определения лучших весов модели и автоматического изменения скорости обучения.

## Структура данных

Директории данных должны быть организованы следующим образом:
```
data_directory/
├── train/
│   ├── class_0/
│   │   ├── image0.jpg
│   │   ├── image1.jpg
│   │   └── ...
│   ├── class_1/
│   │   └── ...
│   └── ...
├── valid/ (опционально)
│   ├── class_0/
│   ├── class_1/
│   └── ...
├── test/ (опционально)
│   ├── class_0/
│   ├── class_1/
│   └── ...
```
Если папки valid или test отсутствуют, данные будут разбиты автоматически из тренировочного набора.

## Использование

Для запуска обучения выполните функцию run(). Весь рабочий цикл автоматизирован и включает подготовку данных, обучение, оценку и сохранение модели.
Пример запуска:
```
from deep_image_classifier import run

# Вызов функции запуска с указанием путей к директориям
run()
```


## Основные функции

- make_dataframes(): подготовка датасета и создание датафреймов.
- make_model(): создание модели transfer learning.
- train(): запуск обучения модели.
- predictor(): оценка модели на тестовых данных.
- save_model(): сохранение обученной модели.
- run(): интерфейс для автоматического запуска процесса обучения и оценки.
